% Observations:
% The PCA based techniques, especially the second one, give significantly better
% denoising results than Bilateral Filtering. This difference can be seen both from
% the RMSD values and from the output images themselves. The PCADenoising2 technique
% gives more than 30% improvement on RMSD of the output image from the original 
% (un-noised) image.

% The second PCA technique based on finding similar patch in a neighborhood gives 
% better result than the vanilla PCA-denoising; but it is much slower!

% We get better edge-preservation with PCA techniques, as compared to Bilateral
% Filtering (see the pattern on the shawl). Also the PCA outputs are smoother.

% Differences b/w PCA and Bilateral Filtering: 
% Bilateral Filtering is based on weighted averaging of the intensities around 
% each pixel, whereas PCA is a reconstruction based technique - the idea being: 
% we can get the effect of denoising by scaling down the eigen-coefficients that 
% "correspond" to noise (whose average squared eigen-coefficients is close to sigma^2). 
% and then reconstructing the image from these modified eigen-coefficients. 

% In order to perform the PCA technique, we need to know the sigma of the added 
% noise, but this is not the case in Bilateral Filtering. 

% There are no parameters to be tuned, in case of PCA-denoising. On the other hand 
% one must tune the values of sigma_S and sigma_I to achieve optimal results 
% with Bilateral Filtering.
